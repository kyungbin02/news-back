# MyApp Backend

Spring Boot 기반의 백엔드 애플리케이션입니다.

## 프로젝트 구조

- **Backend**: Spring Boot + MyBatis + MySQL
- **Frontend**: React (별도 디렉토리)

## 주요 기능

- 사용자 인증 및 관리
- 게시판 기능
- 댓글 시스템
- 관리자 기능
- 문의사항 관리
- 알림 시스템

## 기술 스택

- Java 11+
- Spring Boot 2.7+
- MyBatis
- MySQL
- Gradle

## 실행 방법

1. 데이터베이스 설정
2. `application.yml` 파일 설정
3. 백엔드 실행: `./gradlew bootRun`
4. 프론트엔드 실행: `npm start`

## API 문서

API 문서는 Swagger UI를 통해 확인할 수 있습니다.
